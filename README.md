# 백준 문제 풀이 파일 자동 생성기

백준 온라인 저지(BOJ)의 문제 번호나 URL을 입력받아 해당 문제의 제목을 크롤링하고, 기본 템플릿 코드가 포함된 Python 파일을 자동으로 생성하는 스크립트입니다.

## 기능

- 문제 번호 또는 URL 입력으로 파일 자동 생성
- 백준에서 문제 제목 자동 크롤링
- 파일명, 제목, URL이 포함된 기본 템플릿 생성
- 중복 파일 생성 방지
- 강제 덮어쓰기 옵션 지원
- 향상된 오류 처리 (네트워크 오류 / 파싱 오류 구분)

## 설치

필요한 라이브러리를 설치합니다:

```bash
pip install requests beautifulsoup4
```

## 사용법

### 1. 문제 번호로 파일 생성

```bash
python base.py [문제번호]
```

**예시:**
```bash
python base.py 1000
```

### 2. URL로 파일 생성

```bash
python base.py [문제URL]
```

**예시:**
```bash
python base.py https://www.acmicpc.net/problem/1000
```

### 3. 강제 덮어쓰기

이미 존재하는 파일을 덮어쓰려면 `-f` 옵션을 사용합니다:

```bash
python base.py [문제번호] -f
```

**예시:**
```bash
python base.py 1000 -f
```

## 실행 결과

스크립트를 실행하면 다음과 같은 형식의 Python 파일이 생성됩니다:

**파일명:** `1000.py`

**파일 내용:**
```python
# 1000.py
# A+B
# https://www.acmicpc.net/problem/1000


```

코드 작성을 바로 시작할 수 있도록 빈 줄 2개가 포함되어 있습니다.

### 오류 상황

**사용법이 잘못된 경우:**
```

```

**파일이 이미 존재하는 경우:**
```
⚠️  1000.py 파일이 이미 존재합니다.
💡 덮어쓰려면 -f 옵션을 사용하세요: python base.py 1000 -f
```

**URL에서 문제 번호를 찾을 수 없는 경우:**
```
❌ URL에서 문제 번호를 찾을 수 없습니다.
```

**문제 제목을 찾을 수 없는 경우:**
```
❌ 문제 제목을 찾을 수 없습니다. (올바른 문제 번호인가요?)
```

**네트워크 오류:**
```

```

**파싱 오류:**
```

```

## 주의사항

- 이미 존재하는 파일명과 중복될 경우 `-f` 옵션 없이는 파일이 생성되지 않습니다.
- 인터넷 연결이 필요합니다.
- 백준 사이트의 구조가 변경되면 크롤링이 작동하지 않을 수 있습니다.
